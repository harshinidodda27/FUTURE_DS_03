# Marketing Funnel & Conversion Performance Analysis

## Future Interns Data Science Internship

### Objective
Analyze marketing funnel data to identify conversion patterns, customer behavior, and opportunities to improve campaign performance.

### Dataset
Bank Marketing Dataset (UCI Repository)

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

### Key Insights
- Dataset contains 11,162 customer records.
- No missing values were found.
- Overall conversion rate is 47.38%.
- More customers declined the deposit offer than accepted it.
- Conversion varies across job categories.
- Contact method affects conversion performance.
- Previous successful campaign outcomes lead to higher conversion rates.
- Certain age groups contribute more to conversions.

### Recommendations
- Focus on customer groups with higher conversion rates.
- Use the most effective contact methods.
- Re-target customers who responded positively before.
- Reduce excessive campaign contacts.
- Personalize marketing campaigns based on customer profiles.

### Outcome
Developed a Marketing Funnel & Conversion Performance Analysis report and identified key factors influencing customer conversion.
